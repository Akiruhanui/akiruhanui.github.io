About the Project

An embedded side project for extra learning.

FIND BETTER IMAGES FOR THIS PAGE

This program is similar to an extra "class" that students can take voluntarily, where they learn about embedded programming. This program is usually for those in the electrical or software majors, but I decided that I would try it out too. Although I was able to make it through half the journey (due to an increased course load in my second semester), I still enjoyed learning embedded programming. I plan on continuing my attempts to learn how to program on the board, since I struggled with completing the exercises before.

However, I was able to use my CAD experience to create a simple enclosure for the board that I wanted to be available for everyone. Since the board was a new design introduced this year, there was no case available for it. I decided to try an create a simple design to protect the case and to use it as a potential tool for mounting onto other objects. The design was simple, however a few prototype iterations were required to get the dimensions fully correct. I did not have a 3D printer of my own, however my friend completed the prints for me on his printer. This design included very simple button designs to cover up the push buttons and reset button on the board which do not fall out.
